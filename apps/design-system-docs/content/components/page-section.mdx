## Overview

A layout component for page headers with breadcrumb navigation, titles, and action buttons. Provides consistent spacing and layout patterns across applications.

<InfoCallout type="info" title="When to Use">
Use Page Section at the top of every page to provide context, navigation, and primary actions. It ensures consistent page structure throughout your application.
</InfoCallout>

## Best Practices

<GuidelinesGrid>
	<Guideline type="do" title="Use on every page">
		Include Page Section at the top of every page for consistency. It helps users understand where they are.
	</Guideline>
	
	<Guideline type="do" title="Provide breadcrumbs for deep pages">
		For pages more than 2 levels deep, include breadcrumbs to show the navigation hierarchy.
	</Guideline>
	
	<Guideline type="do" title="Limit primary actions">
		Use only one primary action per page. This helps users focus on the most important action.
	</Guideline>
	
	<Guideline type="dont" title="Don't overload with actions">
		Keep secondary actions to 2-3 buttons. Move less important actions to the overflow menu.
	</Guideline>
	
	<Guideline type="dont" title="Don't use without context">
		Always provide either a pageTitle or breadcrumbs. Users need to know where they are.
	</Guideline>
	
	<Guideline type="dont" title="Don't nest page sections">
		Only use one Page Section per page, at the top level. Don't nest them within other components.
	</Guideline>
</GuidelinesGrid>

## Accessibility

<InfoCallout type="info" title="Breadcrumb Navigation">
Breadcrumbs use proper ARIA navigation landmarks and list structure for screen reader compatibility.
</InfoCallout>

- Breadcrumbs use `<nav>` and `<ol>` for proper semantic structure
- Provide clear, descriptive labels for all actions
- Ensure keyboard navigation works for all interactive elements
- Use `aria-label` for the breadcrumb navigation
- Current page in breadcrumbs should have `aria-current="page"`
- Sidebar toggle should have appropriate ARIA attributes
- Ensure sufficient color contrast for all text and icons
