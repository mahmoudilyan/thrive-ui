## Getting Started

### Installation

```bash
npm install @thrive/ui
# or
pnpm add @thrive/ui
# or
yarn add @thrive/ui
```

### Basic Usage

```tsx
import { Button, Input, Card } from '@thrive/ui';

function MyApp() {
  return (
    <Card>
      <Input placeholder="Enter your name" />
      <Button variant="primary">Submit</Button>
    </Card>
  );
}
```

## Features

### Accessibility First
All components are built with accessibility in mind, featuring:
- Proper ARIA attributes
- Keyboard navigation support
- Screen reader compatibility
- High contrast color schemes

### Design Tokens
Consistent design decisions encoded as tokens:
- Colors with semantic naming
- Typography scales
- Spacing system
- Border radius and shadows

### Dark Mode Support
Toggle between light and dark themes seamlessly:
- System preference detection
- Manual theme switching
- Consistent color schemes across themes
